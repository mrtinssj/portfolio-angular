## 📌 Sobre o projeto

Este portfólio foi criado para apresentar minha trajetória, projetos e habilidades na área de tecnologia.
Desenvolvido com **Angular 17+** e **Angular Material**, com navegação por rotas e design responsivo.

---

## 📄 Páginas

| Página | Descrição |
|--------|-----------|
| 🏠 Início | Apresentação pessoal com cards de tecnologias |
| 👩‍💻 Sobre | Trajetória, interesses e informações pessoais |
| 📁 Projetos | Cards com projetos reais linkados ao GitHub |
| ✉️ Contato | Formulário com validação e envio por e-mail |

---

## ✨ Funcionalidades

- ✅ Navegação por rotas sem recarregar a página
- ✅ Link ativo destacado no menu com `routerLinkActive`
- ✅ Menu lateral responsivo com `mat-sidenav` (hambúrguer no celular)
- ✅ Formulário de contato com `ReactiveFormsModule` e validação
- ✅ Design consistente com identidade visual própria

## 🚀 Como rodar o projeto

```bash
# Instalar dependências
npm install

# Rodar em modo desenvolvimento
ng serve --host 0.0.0.0
```

Acesse em **http://localhost:4200**
